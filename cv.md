## Code Example
Here's my solution to a Codewars problem - "Sum of positive":

```javascript
function positiveSum(arr) {
  return arr
    .filter(num => num > 0)
    .reduce((sum, num) => sum + num, 0);
}

// Alternative solution:
function positiveSum(arr) {
  let sum = 0;
  for (let i = 0; i < arr.length; i++) {
    if (arr[i] > 0) {
      sum += arr[i];
    }
  }
  return sum;
}
```
